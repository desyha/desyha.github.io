Obeng - Online Bengkel

Web aplikasi untuk informasi dan ulasan bengkel.

Automotive workshop listing and review web application.

Project Overview

'Obeng' adalah web-application dimana pengguna dapat mendapatkan dan berbagi informasi mengenai bengkel, serta tempat dimana pemilik bengkel dapat mendaftarkan bisnis mereka untuk mempromosikannya. Projek ini dikembangkan menggunakan framework React-JS pada bagian front-end, NodeJS untuk back-end, dan MongoDB sebagai sistem database.

'Obeng' is a web-application where users can find and share informations about automotive shops around them, and a place workshop owners can list their buisness to promote it. This project was develop using ReactJS as it's front-end framework, NodeJs as it's back-end, and MongoDB as it database system.
